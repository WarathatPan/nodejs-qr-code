# Node.Js-QR-Code

A barebones Node.js app using [Express 4](http://expressjs.com/).

## Running Locally

Make sure you have [Node.js](http://nodejs.org/) installed.

```sh
cd nodejs-qr-code
npm install
cv .env.example .env
set env
npm start
```

Your app should now be running on [localhost:4444](http://localhost:4444/).
